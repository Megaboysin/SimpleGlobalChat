# SimpleGlobalChat
Plugin for Minecraft Spigot/Paper that enables customizable global/local/private/admin chats with compability of placeholderapi and discordsrv
Check branch "master" to see the code
Main language: ru

!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
Check another branch called "master" to see the code
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

Проверенная версия: 1.21.4
Версии которые скорее всего работают 1.21.0-1.21.5
Проверенные ядра: Spigot, Paper

Фишки:
Global chat "!"
Local chat
Admin chat (могут писать и читать только админы)
Ignore/Unignore (работает только для private message)
Appeal Обращение в админ чат
Private Message
Spy (Возможность читать приватные сообщения)
Кастомные сообщения входа, выхода, смерти игроков
Возможность показывать префиксы из luckperms через placeholderapi
Возможность отправки сообщений в дискорд через channels "global", "admin", "appeal"

Если найдете баг или желаемое изменение fell free to write a request!
